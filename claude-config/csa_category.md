# csa_category.md — 동화 카테고리 분류 기준

> 동화 만들기 메뉴의 AI 동화 생성에 활용되는 분류 체계.
> 세 축(구성 배경 / 장르 / 이야기 성격)으로 동화를 정의한다.

---

## 1. 구성 배경 카테고리 (Setting / World)

동화가 펼쳐지는 **세계관·배경**을 나타낸다.
현재 `FairytaleCreateScreen`에서 사용 중인 카테고리 기준.

| 키워드 (ko) | 키워드 (ja) | 이모지 | 설명 |
|-------------|-------------|--------|------|
| 모험 | 冒険 | 🏰 | 성, 던전, 탐험대 등 여정이 중심인 배경 |
| 가족 | 家族 | 👨‍👩‍👧 | 가정, 부모·자녀·형제 관계 중심 배경 |
| 판타지 | ファンタジー | ✨ | 마법 왕국, 이세계, 신화적 세계관 |
| 우정 | 友情 | 🤝 | 친구들 간의 관계가 중심 배경 |
| 동물 | どうぶつ | 🦁 | 동물이 주인공인 의인화 배경 |
| 바다 | 海 | 🌊 | 바닷속, 해적, 섬 등 해양 배경 |
| 우주 | 宇宙 | 🚀 | 우주선, 외계행성, 미래 SF 배경 |
| 마법 | 魔法 | 🧙 | 마법사, 주문, 마법 도구 중심 배경 |
| 숲·자연 | 森・自然 | 🌳 | 요정, 자연 생물, 생태 환경 배경 |

## 2. 장르 (Genre)

동화의 **이야기 전개 방식·분위기**를 나타낸다.
같은 배경이라도 장르에 따라 AI 프롬프트의 톤이 달라진다.

| 키워드 (ko) | 키워드 (ja) | 설명 |
|-------------|-------------|------|
| 클래식 동화 | クラシック童話 | 전통적인 왕자·공주·영웅 구도의 이야기 |
| 전래동화 | 昔話・民話 | 전통 설화 스타일, 교훈 중심의 옛날이야기 |
| 코미디·웃음 | コメディ | 실수·반전·유머로 웃음을 유발하는 이야기 |
| 미스터리 | ミステリー | 수수께끼·단서 찾기·추리가 중심인 이야기 |
| SF·미래 | SF・未来 | 로봇, 인공지능, 미래 기술이 등장하는 이야기 |
| 뮤지컬·노래 | ミュージカル | 노래와 춤이 이야기를 이끄는 형식 |
| 수수께끼·퀴즈 | なぞなぞ | 주인공이 퀴즈·퍼즐을 풀어가는 이야기 |
| 일상·생활동화 | 生活童話 | 평범한 하루에서 특별한 발견을 하는 이야기 |
| 꿈·상상 | 夢・空想 | 꿈속 세계나 상상 속 이야기 |
| 공포·으스스 | こわい話 | 어린이용 가벼운 공포·스릴감 이야기 |

## 3. 이야기 성격 (Theme / Nature)

동화가 전달하는 **핵심 메시지·감정·교훈**을 나타낸다.
AI 동화 생성 시 도덕적 방향성과 감정 톤을 결정하는 축.

| 키워드 (ko) | 키워드 (ja) | 설명 |
|-------------|-------------|------|
| 교훈·도덕 | 教訓・道徳 | 거짓말 금지, 약속 지키기 등 도덕적 메시지 |
| 우정·사이좋기 | 友だち・なかよし | 친구를 사귀고 함께 문제를 해결하는 이야기 |
| 가족 사랑 | 家族愛 | 가족 간의 유대·사랑·그리움이 중심 |
| 용기·도전 | 勇気・チャレンジ | 두려움을 극복하고 새 일에 도전하는 이야기 |
| 성장·자립 | 成長・自立 | 어린이가 스스로 해내며 한 뼘 자라는 이야기 |
| 나눔·배려 | おもいやり・わかちあい | 남을 돕고 함께 나누는 마음을 다루는 이야기 |
| 자기 표현 | 自己表現 | 자신만의 개성을 발견하고 표현하는 이야기 |
| 환경·자연 사랑 | 自然・環境 | 지구와 자연을 소중히 여기는 이야기 |
| 감사·고마움 | 感謝 | 주변 사람과 사물에 감사하는 마음을 전하는 이야기 |
| 문제 해결 | 問題解決 | 어려운 상황을 지혜롭게 해결하는 이야기 |
| 호기심·발견 | 好奇心・発見 | 새로운 것을 탐구하고 발견하는 기쁨을 다루는 이야기 |
| 용서·화해 | ゆるし・仲直り | 다툼 후 용서하고 화해하는 이야기 |

## 4. 조합 예시

동화 한 편은 **배경 1개 + 장르 1개 + 이야기 성격 1~2개**로 구성된다.

| 배경 | 장르 | 이야기 성격 | 예시 제목 |
|------|------|-------------|-----------|
| 숲·자연 | 클래식 동화 | 용기·도전 | 겁쟁이 토끼의 숲속 모험 |
| 우주 | SF·미래 | 우정·사이좋기 | 별빛 행성에서 만난 친구 |
| 동물 | 코미디·웃음 | 나눔·배려 | 실수투성이 아기 곰의 도시락 |
| 왕국·성 | 전래동화 | 교훈·도덕 | 거짓말을 한 왕자의 하루 |
| 바다 | 미스터리 | 호기심·발견 | 사라진 인어 공주의 보물 |

## 5. 개발 활용 가이드

### AI 프롬프트 구성
```
배경(Setting): {구성 배경 카테고리}
장르(Genre): {장르}
주제(Theme): {이야기 성격}
주인공: {사용자 캐릭터 정보}
대상 연령: 3~8세 어린이
언어: 한국어 / 日本語
```

### API 파라미터 예시 (`POST /fairytale/generate`)
```json
{
  "setting": "forest",
  "genre": "classic",
  "theme": ["courage", "friendship"],
  "character_id": "char_abc123",
  "language": "ko"
}
```

### 키워드 enum 값 (권장)

**setting**: `adventure` | `family` | `fantasy` | `friendship` | `animal` | `sea` | `space` | `magic` | `forest` | `kingdom` | `school` | `city`

**genre**: `classic` | `folklore` | `comedy` | `mystery` | `scifi` | `musical` | `quiz` | `daily` | `dream` | `horror`

**theme**: `moral` | `friendship` | `family_love` | `courage` | `growth` | `sharing` | `self_expression` | `environment` | `gratitude` | `problem_solving` | `curiosity` | `forgiveness`
